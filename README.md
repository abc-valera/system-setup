# system-setup

## Initial setup 
1. Update the packages ```sudo dnf update```
2. Install GCC ```sudo dnf install gcc```
3. Install Fish shell ```sudo dnf isntall fish```
4. Install Starship prompt ```curl -sS https://starship.rs/install.sh | sh```
5. Make fish the default shell ```chsh -s /usr/bin/fish```
6. Generate ssh keys: ```ssh-keygen -t ed25519 -C "zoocity14@gmail.com"```, with the names ```github_personal``` and ```github_work```  (for more context https://blog.gitguardian.com/8-easy-steps-to-set-up-multiple-git-accounts/).
7. [Initialize dotfiles](https://github.com/abc-valera/dotfiles)
