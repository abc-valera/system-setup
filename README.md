# system-setup

1. Clone the repository
2. Run the install script `sudo bash ./system-setup/install.sh`
3. Make fish the default shell `chsh -s /usr/bin/fish`
4. Set the system zoom to `1.30` in `Tweaks -> Fonts -> Size -> Scaling Factor`
5. Install Zoom
6. Install UpWork
7. Delete the repository `rm -rf system-setup`
